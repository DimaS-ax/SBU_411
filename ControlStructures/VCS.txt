VCS - Version Control System
SVN, Subversion, Mercury, Git/GitHub
Git != GitHub
Git - это система контроля версий.
GitHub - это облачное хранилище.

Repository - это хранилище исходных кодов.

fetch - скачивает, но не распаковывет изменения с удаленного репозитория;
pull  - скачивает и распаковывает изменения с удаленного репозитория;
push  - отправляет изменения с локального репозитория на удаленный;

commit - контрольная точка, в которую сохраняется текущее состоярие репозитория,
		 и на которую всегда можно вернуться.
		 При создании commit-а обязательно нужно написать commit-message,
		 в котором кратко описывается проделанная работа, сохраняемая в этом commit-e.

branch (ветка) - это последовательность commit-ов
master - основная ветка репозитория
merge - слияние веток
